# WorldBrush

WorldBrush is PaperVTT's custom in-game terrain editing system.

## Scope (Phase 1)
- Radial tool menu on `V`
- Brush radius control + visual preview
- Layer-aware terrain sculpting and paint masks
- Water and snow paint channels

## Attribution
- UX direction and interaction patterns are inspired by TerraBrush.
- This implementation is authored specifically for PaperVTT.
